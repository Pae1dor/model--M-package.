# Odoo 17 ERP — Package M: Retail / Distributor

Mid-tier ERP package for retail and distribution businesses, built on top of Package S (E-commerce).
Excludes POS.

## Key Features

- **Sales Advance**: Quotations, invoices, receipts, credit terms, approval workflows, multi-address, sales tracking & analytics, bilingual printing (TH/EN)
- **Purchasing**: Purchase orders, goods receipt tracking, vendor ledger, RFQ to PO conversion, vendor quotation management, approval workflows
- **Request for Quotation**: RFQ creation, unit-specific data, calendar tracking, purchase history, monthly/yearly summary, purchase analytics by period/product/vendor

## Includes all features from Package S:
- Sales Basic, POS, Inventory & Warehouse

## Tech Stack

- Python 3.10+
- Odoo 17 Community/Enterprise
- PostgreSQL
- XML (Views, Reports, Data)
